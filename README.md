# Rabies Genomic Surveillance in Pakistan

[![GenBank](https://img.shields.io/badge/GenBank-PV491557%2C%20PV641711--PV641717-blue)](https://www.ncbi.nlm.nih.gov/nuccore/PV641717)
![Python](https://img.shields.io/badge/Python-3.8+-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

This repository contains scripts and workflows for the **genomic analysis of rabies virus (RABV) isolates** from Pakistan.

## Project Overview

| Feature | Details |
| :--- | :--- |
| **Organism** | Rabies lyssavirus (RABV) |
| **Source** | Rabid animal samples from Sindh, Pakistan |
| **Genomes Submitted** | 9 whole genomes + 25 partial genes |
| **GenBank Accessions** | PV491557, PV641711–PV641717 |
| **Method** | Whole genome sequencing, assembly, phylogenetic analysis |

## Key Achievements

- ✅ **9 complete genomes** submitted to GenBank
- ✅ **25 partial genes** submitted to GenBank
- ✅ **Phylodynamic analysis** of circulating strains
- ✅ **Co-authored papers** under review:
  - *Infection, Genetics and Evolution* (2026)
  - *PLOS Neglected Tropical Diseases* (2026)

## Repository Contents

- `scripts/` — Python pipelines for assembly, variant calling, and phylogenetics
- `data/` — Sample metadata and GenBank accession numbers
- `results/` — Outputs from the analysis

## Quick Start

```bash
# Clone the repository
git clone https://github.com/durre414622/rabies-genomic-surveillance.git
cd rabies-genomic-surveillance

# Install dependencies
pip install biopython matplotlib numpy pandas

# Run analysis
python scripts/run_analysis.py
```
